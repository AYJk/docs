# 风格基准库 v0.1 —《回收站怪才》

> 目标：先跑出 10 张“风格基准图”，锁定人物、场景、UI、道具的统一方向。后续所有 AI 美术都围绕这些图迭代，避免风格漂移。
>
> 推荐风格：扁平卡通 / 矢量风 / 轻喜剧 / 温暖环保 / 夸张但不脏乱。

---

## 一、全局风格锚点

后续每条 prompt 都建议带上这段风格描述：

### 中文风格锚点

```text
扁平卡通矢量风，适合微信小游戏和抖音小游戏，干净明亮的色彩，轻喜剧感，温暖环保主题，圆润夸张比例，清晰轮廓线，低细节但有记忆点，Q版但不幼稚，现代移动游戏UI美术，2D sprite game art, flat vector illustration, clean shapes, warm lighting, playful, cozy, recycle workshop theme
```

### 英文风格锚点

```text
flat cartoon vector style, mobile casual game art, clean bright colors, warm eco-friendly theme, playful comedy mood, rounded exaggerated proportions, clear outlines, low detail but memorable, cozy recycle workshop, 2D sprite game art, polished mini game visual style, cute but not childish
```

### 负面词

```text
photorealistic, realistic, 3d render, dark horror, dirty disgusting, poverty, beggar, homeless, sadness, messy trash dump, overly detailed, gritty, cyberpunk, violent, ugly, text, watermark, logo, low resolution
```

**重要红线**：视觉上可以“旧物很多”，但不能“脏、惨、贫困、流浪”。主题是“回收站怪才创业”，不是卖惨。

## 二、色彩方向

推荐主色：

| 用途 | 色彩 | 感受 |
|---|---|---|
| 主背景 | 暖米色 / 浅木色 | 亲切、安全、生活感 |
| 回收主题 | 薄荷绿 / 草绿色 | 环保、干净 |
| 科技/改造 | 明黄色 / 橙色 | 灵感、创造、奖励 |
| 主角点缀 | 紫色 / 深蓝 | 怪才、个性 |
| 稀有物品 | 青蓝光 / 金色 | 高级、稀有 |

避免：大面积灰黑、污泥色、过暗垃圾场色调。

## 四、跑图顺序建议

不要一次把 12 条都跑完。按下面顺序迭代：

1. **先跑 01 / 04 / 10**：主角、场景、UI 三大方向先看是否统一。
2. 选出最接近的 1-2 张，作为后续风格参考图。
3. 再跑 02 / 03 / 05 / 06：验证成长线是否一致。
4. 最后跑 08 / 09：确认道具图标能否保持统一。
5. LOGO 最后做，不要太早定。

## 五、验收标准

每张基准图按 5 分制打分：

| 维度 | 标准 |
|---|---|
| 风格统一 | 是否能和其他图放进同一个游戏 |
| 审核安全 | 是否避免贫困、流浪、脏乱、负面价值观 |
| 识别度 | 一眼看出“回收 + 改造 + 怪才” |
| 小屏可读 | 缩到手机屏幕仍能看清主要元素 |
| AI可复制 | 后续能否稳定生成同风格大量图标/角色 |

低于 4 分的图不要进入基准库。

## 六、当前推荐视觉关键词

最终需要固定一组关键词，建议初版为：

```text
flat cartoon vector, cozy recycle workshop, quirky eco genius, warm green and yellow palette, clean outlines, mobile casual game art, cute but not childish, playful comedy, upcycled product design
```

如果跑出来太幼稚：加 `not childish, polished casual game, young adult friendly`。
如果跑出来太脏：加强 `clean, cozy, neatly sorted, optimistic, no dirty garbage`。
如果跑出来太写实：加强 `flat vector, 2D sprite, simple shapes, non-realistic`。
如果道具不够可爱：加 `toy-like, rounded, charming, collectible icon`。
