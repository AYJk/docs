# 白盒原型搭建指南

## 一、代码结构说明

已完成的代码模块：

```
RecycleMaster/assets/scripts/
├── data/                    # 数据层
│   ├── GameConfig.ts       # 游戏配置、枚举、常量
│   ├── MaterialDatabase.ts # 物料数据库（32种物料）
│   └── RecipeDatabase.ts   # 配方数据库（MVP 30个配方）
├── core/                    # 核心逻辑层
│   ├── GridManager.ts      # 格子管理（增删查改）
│   ├── CraftSystem.ts      # 合成系统（第一层升级+第二层配方）
│   └── GameManager.ts      # 游戏管理器（主控制器）
└── ui/                      # UI层
    ├── GameUI.ts           # 游戏UI管理器
    └── GridSlotUI.ts       # 单个格子UI组件
```

## 二、在 Cocos Creator 中搭建场景

### 步骤 1：打开项目

1. 打开 Cocos Dashboard
2. 找到 `RecycleMaster` 项目，点击打开
3. 等待编辑器加载完成

### 步骤 2：创建场景节点结构

在 **层级管理器（Hierarchy）** 中创建以下节点结构：

```
Canvas
├── GameManager (空节点)
├── GameUI (空节点)
│   ├── TopBar (空节点)
│   │   ├── MoneyLabel (Label)
│   │   ├── LevelLabel (Label)
│   │   └── UpgradeButton (Button)
│   │       └── Label (Label) - 显示"升级(成本)"
│   ├── GridContainer (空节点)
│   └── RecipePanel (空节点)
│       └── RecipeListContainer (空节点)
```

### 步骤 3：创建 Prefab（预制体）

#### 3.1 创建 GridSlot Prefab

1. 在 **资源管理器（Assets）** 中右键 → 创建 → Prefab
2. 命名为 `GridSlot`
3. 双击打开，创建节点结构：

```
GridSlot (Node, 添加 UITransform 组件，设置大小 100x100)
├── Background (Sprite, 灰色背景)
├── ItemSprite (Sprite, 初始隐藏)
└── ItemLabel (Label, 初始隐藏)
```

4. 给 `GridSlot` 根节点添加 `GridSlotUI` 组件
5. 在 **属性检查器** 中拖拽关联：
   - `Background` → `GridSlotUI.background`
   - `ItemSprite` → `GridSlotUI.itemSprite`
   - `ItemLabel` → `GridSlotUI.itemLabel`

#### 3.2 创建 RecipeButton Prefab

1. 创建 Prefab 命名为 `RecipeButton`
2. 节点结构：

```
RecipeButton (Button)
└── Label (Label)
```

### 步骤 4：配置主场景组件

#### 4.1 配置 GameManager

1. 选中 `GameManager` 节点
2. 添加 `GameManager` 组件（从脚本拖拽或点击添加组件）

#### 4.2 配置 GameUI

1. 选中 `GameUI` 节点
2. 添加 `GameUI` 组件
3. 在属性检查器中拖拽关联：
   - `Game Manager` → 拖入 `GameManager` 节点
   - `Grid Container` → 拖入 `GridContainer` 节点
   - `Slot Prefab` → 拖入 `GridSlot` Prefab
   - `Money Label` → 拖入 `MoneyLabel` 节点
   - `Level Label` → 拖入 `LevelLabel` 节点
   - `Upgrade Button` → 拖入 `UpgradeButton` 节点
   - `Upgrade Cost Label` → 拖入 `UpgradeButton/Label` 节点
   - `Recipe Panel` → 拖入 `RecipePanel` 节点
   - `Recipe List Container` → 拖入 `RecipeListContainer` 节点
   - `Recipe Button Prefab` → 拖入 `RecipeButton` Prefab

#### 4.3 配置 GridContainer 布局

1. 选中 `GridContainer` 节点
2. 添加 `Layout` 组件
3. 设置：
   - Type: GRID
   - Start Axis: HORIZONTAL
   - Spacing X: 10
   - Spacing Y: 10
   - Padding: 适当设置

#### 4.4 配置 RecipeListContainer 布局

1. 选中 `RecipeListContainer` 节点
2. 添加 `Layout` 组件
3. 设置：
   - Type: VERTICAL
   - Spacing Y: 10

### 步骤 5：运行测试

1. 点击编辑器顶部的 **运行（▶️）** 按钮
2. 应该能看到：
   - 4x4 格子，初始有 8 个废品（玻璃、金属、木料、布料各2个）
   - 顶部显示金币和等级
   - 每隔几秒自动产出新废品
   - 可以拖动相同物品合成升级
   - 右侧显示可制作的配方列表

## 三、测试核心玩法

### 测试第一层合成（升级）

1. 拖动一个 `碎玻璃` 到另一个 `碎玻璃` 上
2. 应该合成为 `玻璃瓶`（Lv2）
3. 继续合成两个 `玻璃瓶` → `玻璃料块`（Lv3）

   - `玻璃瓶×3 + 铁皮×1` = 玻璃瓶吊灯
2. 右侧配方列表会显示可制作的成品
3. 点击配方按钮，消耗材料，获得金币

### 测试升级

## 四、常见问题

### Q1: 编辑器报错找不到模块

**解决**：确保所有 `.ts` 文件都在 `assets/scripts/` 目录下，Cocos 会自动编译。

### Q2: 拖动不生效

**解决**：检查 `GridSlot` 节点是否添加了 `UITransform` 组件，且大小设置正确。

### Q3: 配方列表不显示

**解决**：
1. 检查 `RecipeListContainer` 是否添加了 `Layout` 组件
2. 检查 `RecipeButton` Prefab 是否正确关联

### Q4: 控制台报错

**解决**：打开浏览器控制台（F12），查看具体错误信息。常见错误：
- 组件引用未关联 → 回到场景检查拖拽关联
- 脚本编译错误 → 检查代码是否完整复制
