# 核心循环设计 v0.1 —《回收站怪才》

> 题材：回收站怪才（古怪天才经营废品回收站，把垃圾改造成网红艺术品）
> 玩法：合成为主 + 轻经营 ｜ 平台：微信 + 抖音双端 ｜ 美术：扁平卡通/矢量风

---

## 一、核心循环总览

```
        ┌──────────────── 外环 · 经营（15分钟留存钩子）────────────────┐
        │                                                              │
        │   升级回收站 ← 赚钱 ← 卖成品/展示 ← 改造成品                  │
        │       │                              ↑                       │
        │       ↓                              │                       │
        │   解锁新废品线 / 新配方        ┌──────┴───────┐               │
        │       │                       │  内环 · 合成  │（5秒爽点）    │
        │       ↓                       │              │               │
        │   产出速率↑、格子↑      废品 →[第一层升级]→ 材料 →[第二层配方]→ 成品
        │                               │              │               │
        │   废品来源：①回收站自动产出 ②熟人面具NPC上门卖 ③看广告        │
        └──────────────────────────────────────────────────────────────┘
```

## 二、双层合成系统（核心差异化）

### 2.1 三段式物料链

```
废品(raw) ──第一层:同系升级──> 高级材料(material) ──第二层:跨系配方──> 成品(product)
```

**第一层 · 升级合成（保留经典手感）**
同系废品两两合成，升级为更高级形态。提供"清屏/升级"的基础爽感。

| 废品系 | Lv1 | Lv2 | Lv3（高级材料） | Lv4（稀有材料） |
|--------|-----|-----|----------------|----------------|
| 玻璃系 | 碎玻璃 | 玻璃瓶 | 玻璃料块 | 水晶料 |
| 金属系 | 易拉罐 | 铁皮 | 铁丝卷 | 合金锭 |
| 木料系 | 碎木屑 | 旧木板 | 实木块 | 红木料 |
| 布料系 | 破布条 | 旧衣服 | 布料卷 | 高级面料 |
| 塑料系 | 塑料袋 | 塑料瓶 | 塑料板 | 工程塑料 |
| 纸系   | 废报纸 | 纸箱 | 纸浆块 | 硬纸板 |
| 电子系 | 废电池 | 电路板 | 元件包 | 芯片 |
| 橡胶系 | 旧胎碎 | 橡胶块 | 橡胶垫 | 弹性体 |

> MVP 先做 4 条线（玻璃/金属/木料/布料），V1 扩到 8 条。

**第二层 · 配方改造（差异化核心）**
不同系的高级材料按配方组合成**具象成品**。每个成品独一无二、有外观、能卖钱/展示/进图鉴。

| 成品 | 配方 | 稀有度 |
|------|------|--------|
| 复古台灯 | 玻璃料块×1 + 铁丝卷×1 + 元件包×1 | 普通 |
| 懒人沙发 | 布料卷×2 + 实木块×1 + 橡胶垫×1 | 普通 |
| 摇摇椅 | 实木块×2 + 布料卷×1 | 普通 |
| 风铃 | 玻璃料块×2 + 铁丝卷×1 | 精良 |
| 绿植盆栽 | 塑料板×1 + 纸浆块×1 | 精良 |
| 老式收音机 | 实木块×1 + 元件包×2 + 铁皮×1 | 稀有 |
| 扫地小机器人 | 合金锭×1 + 芯片×1 + 橡胶垫×1 + 电路板×2 | 传说 |

> 成品稀有度（普通/精良/稀有/传说）决定卖价、展示价值、图鉴权重。
> MVP 目标 30 个配方，V1 扩到 80-120 个。

### 2.2 玩家操作手感
- 棋盘/格子界面，拖动两个相同废品 → 合成升级（第一层，经典消除式手感）
- 凑齐配方所需高级材料 → 点击"改造台" → 产出成品（第二层，发现/解谜爽感）
- 双层并行制造策略空间：是继续升级材料，还是凑当前配方？

## 三、经济模型

### 3.1 货币
| 货币 | 来源 | 用途 | MVP |
|------|------|------|-----|
| 软币（回收币 $） | 卖成品 | 升级回收站、收购NPC废品、买图纸 | ✅ |
| 硬币（灵感值 ✦） | 看广告/成就/每日 | 稀有配方、加速、稀有NPC召唤 | ✅ |

> MVP 用"软币 + 广告硬币"双货币即可，不引入第三种付费货币（付费点后置）。

### 3.2 产出与消耗
**产出（钱/废品从哪来）**
- 主动：手动合成 + 凑配方卖成品
- 被动：回收站自动产出废品（挂机/离线收益）← 留存核心
- NPC：熟人面具 NPC 上门卖废品（含稀有材料、开箱感）
- 广告：离线翻倍、稀有召唤

**消耗（钱花到哪）**
- 升级回收站：产出速率↑、格子上限↑、解锁新废品线
- 解锁配方图纸：花软币/硬币买新成品配方
- 加速：跳过合成/改造等待

### 3.3 防通胀
- 卖价随回收站等级线性涨，升级成本指数涨 → 制造"再升一级就够了"的追赶感
- 稀有材料只能靠 NPC/广告获得，限制传说成品产能 → 保住稀有度价值

## 四、NPC 体系（熟人面具）

### 4.1 虚拟 NPC 人设池（打底，解决冷启动）
| NPC | 专精废品 | 稀有倾向 | 人设台词风格 |
|-----|---------|---------|-------------|
| 楼下王大爷 | 通用/纸系 | 量大价低 | 唠叨亲切 |
| 网红小美 | 塑料/包装 | 偶尔稀有 | 时髦跳脱 |
| 收废品老李 | 金属系 | 稳定 | 行家精明 |
| 装修队张工 | 木料/建材 | 量大 | 豪爽 |
| 大学生小赵 | 电子系 | 偶尔稀有 | 社恐省钱 |
| 神秘收藏家 | 稀有材料 | 高稀有高价 | 高冷神秘 |

### 4.2 真实好友面具规则
- 微信：`getFriendCloudStorage` 只读在玩好友的头像+昵称，随机贴到 NPC 皮肤上，台词走通用模板
- 抖音/无数据：全用虚拟 NPC，玩家无感知差异
- **红线**：好友 = 来照顾生意的「贵客」，正面台词（"哟，又给你淘到好东西了！"），绝不丑化成"垃圾贩子"

### 4.3 来访机制
- 定时刷新：每隔 N 分钟来一位 NPC 排队，离线期间攒着，回来一波（强化离线回访钩子）
- 收购交互：NPC "卖"废品，玩家花软币收购（双向交易感，比白送有经营深度）
- 看广告召唤"神秘收藏家"获取稀有材料

## 六、留存钩子

- **离线收益**：回来一波废品 + NPC 排队（每日打开动机）
- **每日配方挑战**：今日指定成品，奖励硬币
- **成品图鉴**：收集驱动（每个成品一张卡）
- **工作室装饰**：成品可摆放展示，养成 + 出片分享
- **周赛季排行榜**：总财富榜 / 图鉴收藏榜 / 周限定改造榜

---

## 七、前 15 分钟节奏（决定次留）

| 时间 | 事件 | 目的 |
|------|------|------|
| 0-1min | 第一次拖动合成升级 | 教学 + 即时爽点 |
| 1-3min | 第一次凑配方做出成品 | 核心爽点"哇原来能这样" |
| 3-5min | 第一次卖货赚钱 + 首个 NPC 上门 | 经营闭环 + 熟人惊喜 |
| 5-10min | 升级回收站，解锁第二条废品线 | 成长感 + 内容扩展 |
| 10-15min | 看到图鉴 / 排行榜 | 建立长期目标 |

## 九、待决策点（标注默认选择）

1. **第一层合成手感**：默认「两两相同合成升级」（经典消除式）。备选「多废品提炼成材料」。
2. **成品去向**：默认「卖钱为主 + 可选摆放展示」。纯卖钱更轻，加展示增养成但加工作量。
3. **货币体系**：默认「软币 + 广告硬币」双货币，付费货币后置。
4. **棋盘形态**：默认「有限格子（满格压力）」。备选「无限滚动列表」。
